# VPC CRM - Elevando seu Potencial de Vendas

Landing Page profissional para o sistema de CRM da **Vertical Parts**, focado em automação de vendas e gestão de leads.

## 🚀 Tecnologias
- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid, Animações)
- JavaScript Vanilla (Interações e Scroll Reveal)
- Google Fonts (Red Hat Display)

## 🎨 Identidade Visual
O design segue a identidade industrial da Vertical Parts:
- **Cores**: Amarelo Vibrante (`#FFCD00`) e Preto Profundo.
- **Estética**: Industrial, técnica e de alto impacto.
- **Tipografia**: Red Hat Display para um visual geométrico e moderno.

## 📁 Estrutura do Projeto
- `index.html`: Estrutura da landing page.
- `style.css`: Estilização completa e responsiva.
- `main.js`: Interações dinâmicas.
